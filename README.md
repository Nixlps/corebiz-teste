## Tecnologias 

ReactJS 18.2.0
SASS 1.64
Webpack 5.88

## Setup 

Para rodar o projeto, comece instalando as dependências 

### $npm install

Em seguida, para abrir a aplicação no modo desenvolvimento digite

### npm run dev

Para gerar o bundle de produção digite

### npm run build
